# playGround
이것저것 실험실입니다.

```bash
├── src.main.java.com.quid
│   ├── DockerCompose
│   │   └── 간단 설정 정리
│   ├── ELK
│   ├── Java
│   │   └── java의 다양한 기능들 정리(RxJava, Lambda 등)
│   ├── Kotlin
│   ├── openAi
│   │   └── openAi를 이용한 간단한 텍스트 IO
│   └── Spring
│       ├── config
│       ├── diary
│       │    └── mySQL 인덱싱, 벌크인서트 등을 테스트해보기 위한 폴더
│       └── FanOutTimeline
│            └── 팔로우한 사람의 글을 빠르게 가져오기 위한 테스트
│                카프카를 사용한 비동기 처리
└─── Readme.md
```
